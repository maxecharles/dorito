# DORITO

DORITO is an image reconstruction framework designed to interface with [AMIGO](https://github.com/LouisDesdoigts/amigo), an end-to-end differentiable modelling pipeline for Aperture Masking Interferometer (AMI) data from the James Webb Space Telescope (AMI).
